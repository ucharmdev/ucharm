# Modules

ucharm includes 50+ native modules implemented in Zig for maximum performance and minimal binary size.

## TUI Components

| Module | Description |
|--------|-------------|
| [charm](/docs/modules/charm) | Boxes, tables, progress bars, spinners, status messages |
| [input](/docs/modules/input) | Interactive prompts: select, multiselect, confirm, password |
| [term](/docs/modules/term) | Terminal control: size, cursor, raw mode |
| [ansi](/docs/modules/ansi) | ANSI escape codes for colors and styles |

## Networking

| Module | Description |
|--------|-------------|
| [fetch](/docs/modules/fetch) | HTTP/HTTPS client with built-in TLS |
| http.client | Low-level HTTP client |

## Data Formats

| Module | Description |
|--------|-------------|
| json | JSON encoding/decoding |
| csv | CSV reading/writing |
| toml / tomllib | TOML parsing |
| xml.etree.ElementTree | XML parsing |
